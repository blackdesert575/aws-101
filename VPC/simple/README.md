# simple

* Make some example from Terraform VPC module example: simple
    * There is a public and private subnet created per availability zone in addition to single NAT Gateway shared between all 3 availability zones.

## Quick Start

```shell
$ terraform init
$ terraform plan
$ terraform apply
```
